Website info

This site was written from scratch, half out of being too prideful to use a template, and half out of wanting to learn to build sites. It likely breaks many many best practices in web design, but its functional.

I use exljbris Font Foundry's handsome Museo Slab font for heading text and Sebastian Kosch's crisp Crimson Text font for everything else. I don't know if these font choices obey whatever typography fundamentals in web design there are. I just eyeballed it, thinking they look nice :)

Written with Sublime Text
