Discord Reply for Pebble

A handy applet to reply to Discord DMs for the (sadly defunct) Pebble watch.

Download it on the Rebble Store

Because this applet utilizes Pebble's voice features, and the official Pebble voice recognition endpoint is dead, you'll need to subscribe to Rebble web services to send messages (only 3 bucks a month, which I think is pretty reasonable)
